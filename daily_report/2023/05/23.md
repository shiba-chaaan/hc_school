## わかったこと

- Golang の良さ
  - シンプルで可読性の高いコードが書ける（誰が書いても同じような記述になる）
  - 学習コストが低い（シンプルな記述）
  - コンパイルが早い、コンパイル済み速度も C に迫る実行速度のは良さ
  - 標準パッケージが多い
  - 安全性が高い（静的型付け言語で、タイプセーフでありメモリセーフである）
  - 非同期処理が簡単に使える
  - 拡張性が高い
  - C 言語の実行速度と Python などの開発速度を合わせたようないいとこ取り言語
- Golang 基本文法
  - 変数、定数、演算子、関数、制御構文
